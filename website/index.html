Install in minutes

Android

  1. Tap Download for Android.
  2. Open the APK. If asked, allow installs from this browser/source.
  3. Open TechRen EDU and sign in.

Windows

  1. Tap Download for Windows.
  2. Run TechRenEDU-setup.exe.
  3. Finish the wizard — a TechRen EDU shortcut appears on your Desktop and Start Menu.

If Smart App Control blocks the setup (only “Okay”, no Run anyway): open Windows Security → App & browser control → Smart App Control and set it to Off, then run the setup again. (Unsigned school apps need this until a code-signing certificate is used.)

Alternative: download the portable zip, unzip, and run techren_edu.exe (same Smart App Control rule applies).

Need help signing in? Contact your school administrator.

Native install only — not a browser “Add to Home Screen” app.
