Install in minutes

Android

  1. Tap Download for Android.
  2. Open the APK. If asked, allow installs from this browser/source.
  3. Open TechRen EDU and sign in.

Windows

  1. Tap Download for Windows.
  2. Unzip the folder.
  3. Run techren_edu.exe and sign in.

Building the Windows zip on this PC requires Visual Studio with Desktop development with C++.

Need help signing in? Contact your school administrator.

Native install only — not a browser “Add to Home Screen” app.
